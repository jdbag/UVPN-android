# U VPN – آمن وسريع 🛡️

تطبيق VPN عالمي مجاني وآمن مبني بالويب.

![U VPN Logo](logo.svg)

## ✨ المميزات

- 🔒 **تشفير كامل** – حماية بيانواتك بالكامل
- ⚡ **سرعة عالية** – بروتوكول WireGuard
- 🌍 **15+ دولة** – خوادم في كل أنحاء العالم
- 🚫 **حجب الإعلانات** – تصفح نظيف
- 🛡️ **Kill Switch** – حماية تلقائية
- 📱 **تصميم متجاوب** – يعمل على كل الأجهزة
- 🆓 **مجاني** – لا يلزم تسجيل

## 🚀 تشغيل التطبيق

افتح `index.html` مباشرة في المتصفح أو استضفه على أي خادم ويب.

```bash
# باستخدام Python
python3 -m http.server 8080

# باستخدام Node.js
npx serve .
```
