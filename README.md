# U VPN – Fast & Secure 🛡️

A free global VPN app built with web technologies.

![U VPN](logo.svg)

## ✨ Features

- 🔒 **Full Encryption** – Your data is fully protected
- ⚡ **High Speed** – WireGuard protocol
- 🌍 **15+ Countries** – Servers worldwide
- 🚫 **Ad Blocker** – Clean browsing
- 🛡️ **Kill Switch** – Auto protection
- 📱 **Responsive Design** – Works on all devices
- 🆓 **Free** – No registration required
- 📍 **Real IP Detection** – Shows your actual public IP

## 🚀 Run the App

Open `index.html` directly in a browser or host it on any web server.

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve .
```
